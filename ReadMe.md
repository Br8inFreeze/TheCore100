# The Core 100
这份题单整理自我近些年的找工历程，如果我下周有面试，这会是我这周集中复习的100题。我之前也刷过一些网上热门的题单例如hot 100, grind 75等等，根据这些题单的痛点和我的真实面试体验，我做出了如下改进：
1. 题目并非热门，至少在北美SDE/MLE的找工市场里并不是热门面试题。这一类的题大家在刷这些题单的时候也很容易通过leetcode自带的Companies频次统计来看出，也很少在1p3c的面经里见到。所以我结合了自己的面试经历和浏览过的面经，选出了真正在北美求职面试中的高频题
2. 题目缺少代表性，这一类题通常难以举一反三，考察的东西很偏，重复练习的价值并不高。当然如果你在准备特定公司的面试时，会在面经中看到一部分这样的题，当然值得突击准备。但我认为不适合放在一份精选的题单中。
3. 覆盖的知识范围有偏差，达不到essential的标准。举例来说，有一些题单完全不含DP或是图论这样的考点，然而这些仍然是面试中的热门考点；另一些题单又包含了一些高难而冷门的考点，例如TreeMap或者线段树，我觉得没有必要。而我的这份题单cover的是面试中最主流最常见的考点。

这里是我选出的100题,数字为leetcode题号，当然有一些题值得用多种解法来做，有一些题有若干follow-up最好一并解决，过一遍的话实际的题量会略大于100。另一方面，个人选出的这100题难免片面和存在疏漏，也欢迎大家补充。
